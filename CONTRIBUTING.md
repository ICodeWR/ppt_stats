# 贡献指南

感谢你对 ppt-stats 的关注！欢迎提交 Issue、PR 或参与讨论。

## 如何贡献

### 报告 Bug

1. 在 Issues 中搜索是否已有类似问题
2. 提供清晰的标题和描述
3. 附上复现步骤、期望结果和实际结果
4. 标注运行环境（Python 版本、操作系统）

### 提交代码

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/xxx`
3. 提交前确保代码可正常运行
4. 提交 PR 时说明改动内容和原因

## 开发环境

```bash
git clone <repo-url>
cd ppt-stats
uv sync
```

## 代码规范

- 使用 Python 3.9+ 语法
- 函数和模块添加中文 docstring
- 保持代码简洁，避免过度抽象

## 沟通

- 在 Issue 中讨论功能需求
- 大型改动请先开 Issue 讨论再动手
